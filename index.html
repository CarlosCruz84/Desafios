Esse é um breve paragráfo para um teste feito pelo chatgpt para eu treinar html e css

Cada um destes parágrafos está em uma div diferente, e se a div maior que engloba esses três textos não existisse, todos os itens seguiriam a regra da div container que tem o fundo preto e engloba a foto e os textos. Por padrão, o display: flex determina que todos os itens dentro dela fiquem na mesma linha. Para inverter isso e deixar os itens empilhados em uma coluna, usamos a propriedade flex-position: column.
